# Simple Allow Copy â€” Chrome Extension

A lightweight Chrome extension that re-enables **right-click**, **text selection**, and **copy** on websites that block these features using JavaScript.

## Features
- Enable right-click on restricted pages  
- Allow text selection and copying  
- Automatically removes common copy-blocking scripts  
- Simple, no-permissions design

## Installation (Developer Mode)
1. Download or clone this repository.
2. Opent he extension and the other folder inside it
3. Open **chrome://extensions/** in your Chrome browser.  
4. Enable **Developer mode** (top-right toggle).  
5. Click **Load unpacked**.  
6. Select the extension folder.

This restores default browser behavior on most sites.

## Notes
- Some websites use advanced copy-protection methods that cannot be bypassed.  
- Use this extension responsibly and follow the Terms of Service of any website you visit.
